日付入りディレクトリへのデータ保存スクリプト

Linux上での結果を残しておくための簡単なスクリプト例をメモしておく．

いくつかの結果ファイルを日付の入ったディレクトリに保存するスクリプト．

#!/bin/bash

now=`date +%y%m%d_%H%M%S`
dname=results_$now
mkdir $dname
cp some result files $dname/
echo some result files
echo "These files are Saved at " $dname
