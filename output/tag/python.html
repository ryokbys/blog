Pythonで可変数の引数の取り方 *args, **kwds

Posted on Tue 22 March 2016 in misc • Tagged with python

関数の引数に *args や **kwds を指定した場合の挙動をいつも忘れてしまうので，例と共に書いておく．

def f(*args, **kwds):
    print args
    print kwds

上のような関数は， args は変数名なしの引数をタプルとして受け取り， kwds は変数名付きで辞書型として受け取る．

f(1,2,3,4,a='a',b='b')

{'a': 'a', 'b': 'b'}
